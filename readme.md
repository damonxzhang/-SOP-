# 维修 SOP 系统 (Maintenance SOP System)

> **当前版本：V1.2.20260212.004**  
> **更新内容：新增侧边栏“个人信息”中心，支持查看工号、部门、安全设置及个人操作动态；优化了导航栏用户交互。**

这是一个专为高精密制造场景设计的数字化维修作业规程管理平台，旨在通过标准化的作业流程（SOP）提升维修效率，降低人为差错，并实现维修过程的全数字化追踪。

## 核心功能需求实现

### 1. PDA端 (Mobile-First Experience)
- **实时报修管理**：
  - 查看待处理报修列表。
  - **详细报修看板**：点击查看故障码、设备信息（SN/型号）、故障分类及报修类型。
  - **现场证据链**：支持展示报修时上传的现场保修图片，帮助工程师预判故障。
- **标准化 SOP 执行**：
  - **多维度步骤指导**：每个维修步骤均包含**操作说明**、**判断方法**、**步骤描述**及高亮的**安全提示**。
  - **多媒体辅助**：支持图片、视频及 PDF 文档的实时查阅。
  - **交互式反馈**：支持在执行过程中对特定步骤进行提问或反馈新发现的故障点。
- **视觉强化**：针对关键核查项采用高亮（加粗红字）设计，确保核心风险点不被忽略。

### 2. 管理员端 (Centralized Control)
- **布局优化**：**强制侧边栏导航**，在所有屏幕尺寸下均保持左侧固定布局，提升后台管理体验。
- **统计看板 (BI Dashboard)**：
  - 监控平均修复时间 (MTTR)、SOP 依同率等核心指标。
  - 故障趋势分布与预测模型。
- **标准 SOP 库管理**：
  - **深度集成资源库**：在编辑 SOP 步骤时，支持**本地上传**或**从资源库选择**。
  - **智能检索引用**：从资源库引用时支持通过**名称或标签**进行实时搜索，实现资源的快速复用。
- **多媒体数字化资料库**：
  - **统一管理**：集中存储和管理维保过程中的图片、视频教学及技术文档。
  - **标签检索**：支持对资料进行多维标签（Tags）标注，实现快速筛选与定位。
  - **全生命周期管理**：支持资料的在线上传、实时预览（图片/视频/文档）、属性编辑及安全删除。
- **现场提问与闭环**：
  - 实时接收工程师反馈，支持专家远程下达回复指令，实现问题的闭环管理。
- **个人信息中心**：
  - **账户概览**：查看姓名、工号、所属部门及入职时间。
  - **安全与偏好**：支持修改密码、管理两步验证等安全设置。
  - **动态追踪**：记录工程师最近在系统内的关键操作记录。
- **用户与权限矩阵 (RBAC)**：
  - 基于角色的访问控制，定义资产操作红线与权限颗粒度。

## 技术栈
- **Frontend**: React 19 + TypeScript + Vite
- **Styling**: Tailwind CSS (移动端适配与响应式布局)
- **Icons**: Lucide React
- **Charts**: Recharts (数字化统计)
- **Animations**: Tailwind Animate

## 快速开始
1. **安装依赖**：`npm install`
2. **启动开发服务器**：`npm run dev`
3. **构建生产版本**：`npm run build`
